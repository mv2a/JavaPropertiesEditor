Java Properties Editor (JFace TableViewer)

Eclipse plug-in for editing Java properties files using JFace TableViewer.

View on GitHub Download .zip Download .tar.gz

Installation

Click "Help -> Install new software" and paste update site URL:
http://mv2a.github.io/JavaPropertiesEditor/updates/

Usage

A new entry named "Table Properties Editor" will be available on *.properties files under the "Open With" context menu entry, Project or Package Explorer.
